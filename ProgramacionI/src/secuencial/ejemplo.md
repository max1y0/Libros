# Ejemplo
Hacer un programa en python que calcule (aproximadamente) cuántos días pasaron desde que una persona nació.

Encaremos el análisis juntos: 

**¿Qué queremos hacer?**

Un algoritmo para calcular los días desde que nació de una persona

**¿Cómo lo vamos  a hacer?**

Si sabemos la edad de una persona, podemos multiplicar ese número por 365 para tener un aproximado de la cantidad de días de vida.

Entonces:
1. Necesitamos preguntarle al usuario cuál es su edad y guardarla en una variable. 
2. Luego necesitamos otra variable para guardar el resultado final.
3. A esa variable le tenemos que asignar ```edad * 365``` y 
4. Mostrar el resultado por pantalla
